Network Tech Documentation
==========================

Network Tech is an add-on (Package) for Sublime Text providing Cisco configuration highlighting, auto completions and more.

Quick Start
-----------

1. `Install Sublime Text <https://www.sublimetext.com/>`_

2. `Activate Package Control <https://packagecontrol.io/installation>`_ in Sublime Text

3. Install the package:
      :menuselection:`Tools --> Command Pallet`
      
      ‚å® :kbd:`Package Control: Install Package`

      ‚å® :kbd:`<enter>`
      
      ‚å® :kbd:`Network Tech`

      ‚å® :kbd:`<enter>`

4. Enable the syntax highlighting
      :menuselection:`Tools --> Command Pallet`

      ‚å® :kbd:`Set Syntax: Cisco IOS`

      ‚å® :kbd:`<enter>`
